1. 소스트리 설치
	에러나 묻는거 뜨는건 다 아니오/취소
2. 컴퓨터 내 빈 폴더 생성 (완전히 빈 폴더)
3. 소스트리 new tab에서 Clone
4. 소스 경로 / URL (저장소경로) : 깃 허브 readme file 있는 곳 code 누르면 https 보이는 링크 복사 후 붙여넣기
5. 목적지 경로 : 컴퓨터 내 빈 폴더 주소 (선택)
6. 밑에 클론 버튼 누르기
7. 생성후 오른쪽 위에 설정 클릭
8. origin 경로를 https://본인 깃허브 이름:토큰@github.com/본인 깃허브 이름/깃허브 이름.git -> 확인
9. 내 컴퓨터에 없는 파일을 받는 것은 pull
10. 내 컴퓨터에 클론 폴더에 새 파일을 복사붙여넣기 등 새로운 이벤트가 발생하고 깃허브에 올리고 싶을때는 push
-> 클론 폴더에 파일/폴더 를 넣고
-> 소스트리에서 왼쪽 위 커밋 클릭 (자동으로 소스트리가 인식)
-> 오른쪽 밑에 커밋 클릭 (메시지는 작성 해도 되고 안해도 됨)
-> push 클릭
끝
